# portfolio

Kayla Heath's personal website / portfolio.

Built with [Vite](https://vite.dev/), [React](https://react.dev/), and TypeScript.

## Development

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build
npm run lint     # lint
```

## Assets

The animated pixel cat sprites are **paid/licensed art and are not committed** to this
repo (see `.gitignore` and [`src/assets/sprites/README.md`](src/assets/sprites/README.md)).
To run with the cat locally, add the purchased spritesheets to `src/assets/sprites/`.
